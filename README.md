## What’s inside these repos?

### tr-adres (PHP 8+ Composer Library)
SQL import olmadan, hazır JSON API ile Türkiye adres zinciri (il → ilçe → mahalle → yol).
- Composer ile hızlı kurulum
- Cache + CLI + testler + checkout demo

➡️ Repo: https://github.com/mebularts/tr-adres

### Copy-Telegram-Content (Telegram Content Ops)
Birden fazla Telegram kanal/grubunu takip edip yeni içerikleri hedef kanal/gruba otomatik paylaşma.
- JSON ile kanal listesi
- Medya destekli paylaşım
- Tekrarları engelleme (geçmiş takibi)

➡️ Repo: https://github.com/mebularts/Copy-Telegram-Content

### Youtube-Content-on-Telegram (YouTube → Telegram Pipeline)
Seçtiğin YouTube kanallarındaki yeni videoları çekip Telegram’da otomatik paylaşma.
- YouTube API ile metadata
- Video indirme + gönderme
- Gönderim geçmişi ile tekrarları engelleme

➡️ Repo: https://github.com/mebularts/Youtube-Content-on-Telegram

### Automatic-Language-Support (Auto i18n for PHP/HTML)
Mevcut PHP/HTML projelere otomatik dil desteği ekleme.
- Metinleri JSON’larda yönetme
- Modern dil switcher
- Sorunsuz entegrasyon yaklaşımı

➡️ Repo: https://github.com/mebularts/Automatic-Language-Support

### location-find-prank (Experimental / Fun)
Deneysel ve eğlence amaçlı küçük bir proje.
➡️ Repo: https://github.com/mebularts/location-find-prank

> Not: Projeleri lütfen etik şekilde kullanın. Mahremiyeti ihlal eden, izinsiz takip/izleme gibi amaçlar desteklenmez.
